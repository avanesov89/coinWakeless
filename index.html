Первая настройка

Вставьте сюда конфиг вашего проекта Firebase (объект firebaseConfig из консоли Firebase — Project settings → Your apps → SDK setup). Можно вставить как есть, в кавычках или без — приложение разберёт само.

Где взять конфиг: console.firebase.google.com → создать проект → в разделе Build открыть Firestore Database → Create database (test mode подходит для старта) → включить Authentication → Sign-in method → Anonymous → в настройках проекта (шестерёнка) добавить веб-приложение и скопировать объект конфигурации.

Новый расчёт распределения

Отметьте походы, которые войдут в этот расчёт (по умолчанию — только ещё не распределённые). Весь ещё не распределённый дроп подтягивается автоматически — без исключений. Дроп, привязанный к конкретному походу, делится между участниками именно этого похода (поровну, либо с учётом коэффициента вклада, если он был задан для похода — см. вкладку «Походы»); непривязанный дроп делится поровну между всеми отмеченными походами, а дальше — так же между их участниками. Итог по игроку — сумма его долей по тем походам, где он реально был, поэтому у ходившего 2 раза из 4 сумма считается только из этих двух дней.
